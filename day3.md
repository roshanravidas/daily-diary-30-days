# Day 3 - 27 June 2026
## Topic Learned
- CSS Flexbox (justify-content, align-items, flex-direction)
- CSS Grid (rows, columns, gap)
- Responsive design basics

## Tasks Done
- Created a simple webpage layout using Flexbox
- Practiced aligning items horizontally and vertically
- Built a 2-column grid layout with header, sidebar, and footer
- Tested responsiveness by resizing browser window

## Challenges
- Confused between Flexbox vs Grid usage
- Struggled with aligning items perfectly in grid cells

## Solutions
- Learned rule of thumb: Flexbox for one-dimensional layouts, Grid for two-dimensional
- Used `gap` property in Grid instead of margins for cleaner spacing
- Practiced with Chrome DevTools to visualize layout changes

## Reflection
- Flexbox and Grid make complex layouts much easier compared to plain CSS
- Responsive design is essential for modern web apps
- Feeling more confident about structuring pages for MERN projects
